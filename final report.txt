3. 디자인 계획
1) 현재 정신병원에 음압병실이 있는 병원들의 수와 문제점 
현재 정신병원에 음압병실이 있는 병원의 수 : 국립정신건강센터(1곳) → 4월기준 [문제점]하고 신속한 진단이 어려움
→ 의료진이 환자와 소통하기 위해 거쳐야하는 단계가 많아서 시간이 지체됨
→ 환자들과 직접적으로 소통하기 어렵기 때문에 정확
→ 확진자를 초기에 발견하기 어려움
→ 환자들의 증상이 명확하지 않기 때문에 정확한 진단에 어려움이 있음(면역력이 약해 무증상 환자들이 많음)
→ 환자들 수에 비해 의료진들의 수가 부족하기 때문에 진단하는데 시간이 많이 소요됨
→ 방호복으로 인해 서로 소통이 잘 되지 않음. (물리적인 소통의 어려움)
→ 정상적인 소통조차 되지 않는 환자도 있어 소통이 어려움
→ 자신의 상태를 잘 설명하지 못하는 환자들이 많음

3) design description 
→ 환자들과 의료진들과의 원활한 의사소통을 위해 환자가 의사를 명확하게 표현할 수 있어야 한다.

4) 환자의 요구사항이나 의사표현에는 머리 아픔, 기침이 심해짐, 화장실 동행 필요 등이 있을 수 있으며 환자가 어떤 사항을 선택하면, 환자의 호실 정보와 환자가 선택한 사항이 의료진에게 전달된다.
